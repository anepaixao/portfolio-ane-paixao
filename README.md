# Ane Portfolio

Portfólio profissional desenvolvido com React, TypeScript, Vite e Tailwind CSS, com foco em desenvolvimento de software, front-end/mobile, produto digital e inteligência artificial aplicada.

## Objetivo

Este projeto apresenta minha trajetória, projetos, experiências e habilidades de forma moderna, responsiva e alinhada ao meu posicionamento profissional.

O portfólio funciona como uma landing page profissional para apoiar minha presença no GitHub, LinkedIn e currículo, destacando minha evolução em tecnologia, produto digital e IA aplicada.

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Seções do site

- Início
- Sobre mim
- Projetos
- Experiências
- Habilidades
- Currículo
- Contato

## Projetos em destaque

### Macuco Entregas

Aplicativo mobile de delivery local voltado à modernização do comércio em Buerarema-BA, com foco em experiência mobile, regras de negócio, MVP e organização de produto.

### FinanceirAne

Assistente financeira pessoal em desenvolvimento, focada em apoiar o registro, organização e acompanhamento de gastos, conectando automação, dados, experiência conversacional e educação financeira.

### IA na Robótica Educacional

Projeto de iniciação científica sobre o uso de inteligência artificial na robótica educacional, investigando potencialidades, desafios e aplicações no contexto da educação STEM.

## Como rodar localmente

```bash
git clone https://github.com/anealmeida191/ane-portfolio.git
cd ane-portfolio
npm install
npm run dev
```

Depois, acesse o endereço exibido no terminal.
