# Ane Paixão | Portfólio Profissional

Portfólio profissional desenvolvido com React, TypeScript, Vite e Tailwind CSS, com foco em desenvolvimento de software, front-end/mobile, produto digital e inteligência artificial aplicada.

Este projeto apresenta minha trajetória, projetos, experiências e habilidades de forma moderna, responsiva e alinhada ao meu posicionamento profissional.

O portfólio funciona como uma landing page para apoiar minha presença no GitHub, LinkedIn e currículo, destacando minha evolução técnica e meu interesse pela construção de soluções digitais com propósito, tecnologia e visão de usuário.

---

## Objetivo

Apresentar de forma clara e profissional meus principais projetos, experiências e habilidades, conectando desenvolvimento de software, produto digital e inteligência artificial aplicada.

O design e a estrutura do projeto foram pensados para traduzir meu posicionamento profissional: construir soluções úteis, acessíveis e bem organizadas a partir de problemas reais.

---

## Tecnologias utilizadas

Este projeto foi construído com ferramentas modernas do ecossistema front-end:

- **React** — biblioteca para construção da interface
- **TypeScript** — tipagem estática para maior segurança no código
- **Vite** — ambiente de desenvolvimento rápido e moderno
- **Tailwind CSS** — estilização utilitária e responsiva
- **Framer Motion** — animações leves e interativas
- **Lucide React** — biblioteca de ícones

### Macuco Entregas

Aplicativo mobile de delivery local voltado à modernização do comércio em Buerarema-BA, com foco em experiência mobile, regras de negócio, MVP e organização de produto.

### FinanceirAne

Assistente financeira pessoal em desenvolvimento, focada em apoiar o registro, organização e acompanhamento de gastos, conectando automação, dados, experiência conversacional e educação financeira.

### IA na Robótica Educacional

Projeto de iniciação científica sobre o uso de inteligência artificial na robótica educacional, investigando potencialidades, desafios e aplicações no contexto da educação STEM.

## Como rodar localmente

Para clonar e executar o projeto localmente:

```bash
git clone https://github.com/anepaixao/portfolio-ane-paixao.git
cd portfolio-ane-paixao
npm install
npm run dev
```

Depois, acesse o endereço exibido no terminal, geralmente:

```bash
http://localhost:5173
```
