# Ane Portfolio

Portfólio profissional desenvolvido com React, TypeScript e Tailwind CSS.

O projeto apresenta minha trajetória, projetos, experiências e habilidades na área de desenvolvimento de software, produto digital e IA aplicada.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Seções do portfólio

- Início
- Sobre mim
- Projetos
- Experiências
- Habilidades
- Currículo
- Contato

## Objetivo

Este portfólio tem como objetivo apresentar minha evolução profissional, meus projetos e meu posicionamento como desenvolvedora front-end/mobile em formação, com interesse em produto digital e inteligência artificial aplicada.
