# Portfolio Ane Pixão
Portfólio profissional desenvolvido com **React**, **TypeScript**, **Vite** e **Tailwind CSS**, com foco em apresentar minha trajetória, projetos, habilidades e posicionamento profissional nas áreas de **desenvolvimento front-end/mobile**, **produto digital** e **inteligência artificial aplicada**.

## Sobre o projeto

Este projeto foi criado como uma landing page profissional, moderna e responsiva, funcionando como uma extensão visual do meu currículo, LinkedIn e GitHub.

O objetivo é apresentar, de forma clara e recrutador-friendly, minha atuação como estudante de Ciência da Computação e desenvolvedora em formação, destacando projetos práticos, experiências acadêmicas e interesses profissionais relacionados à construção de produtos digitais.

## Objetivo

Construir um portfólio profissional que comunique:

- minha trajetória em tecnologia;
- meus principais projetos;
- minhas habilidades técnicas;
- minha atuação com produto digital;
- meu interesse em IA aplicada;
- minha busca por oportunidades de estágio e posições iniciais em tecnologia.

## Projetos em destaque

### Macuco Entregas

Aplicativo mobile de delivery local voltado à modernização do comércio em Buerarema-BA, desenvolvido como MVP para conectar lojas, consumidores e pedidos em uma experiência simples e acessível.

**Principais pontos:**

- desenvolvimento front-end/mobile;
- estruturação de telas, componentes e fluxos;
- definição de escopo do MVP;
- documentação de regras de negócio;
- liderança e organização da equipe.

### Financeirane

Assistente financeira pessoal em desenvolvimento, criada para apoiar o registro, organização e acompanhamento de gastos, conectando automação, dados, experiência conversacional e educação financeira.

**Principais pontos:**

- organização de informações financeiras;
- experiência conversacional;
- automação de registros e acompanhamento;
- estruturação de dados;
- projeto em desenvolvimento.

### IA na Robótica Educacional

Projeto de iniciação científica sobre o uso de inteligência artificial na robótica educacional, investigando potencialidades, desafios e aplicações no contexto da educação STEM.

**Principais pontos:**

- pesquisa acadêmica;
- análise comparativa de ferramentas;
- escrita técnica;
- comunicação científica;
- aplicações de IA na educação.

## Como rodar localmente

```bash
git clone https://github.com/anepaixao/ane-portfolio.git
cd ane-portfolio
npm install
npm run dev
```

Depois, acesse o projeto no navegador pelo endereço exibido no terminal, geralmente:

```bash
http://localhost:5173
```

## Build para produção

```bash
npm run build
```

## Preview do build

```bash
npm run preview
```
